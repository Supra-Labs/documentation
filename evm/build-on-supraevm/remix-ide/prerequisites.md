# Prerequisites

* **A** [**StarKey Wallet**](https://starkey.app/) is installed
* A wallet account with testnet funds
* [**Supra MultiVM RPC**](../../network-information.md) is already selected inside StarKey Wallet

{% hint style="info" %}
Request the testnet fund [here](https://rpc-multivm.supra.com/docs/#/operations/evm_faucet).
{% endhint %}

These tools allow Remix to connect to the SupraEVM network and sign transactions using StarKey Wallet.
